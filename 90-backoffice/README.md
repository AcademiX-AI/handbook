## 弊社から送付する請求書の作り方
まずは、写真のように”入力情報”から必要な箇所（黄色枠内）を入力。
その後、必要なシート（見積書、請求書、納品書、領収書）をクリックすると自動生成されているため、PDFにて保存。
その後、Googleドライブの対象月に保存。※命名規則　「書類種別_日付_クライアント名　」 e.g.請求書_202208_ティンバーズ様
